## 2. 注册

**功能说明：**
- 用户未注册时，在引导页，点击【进入聊聊】按钮，进入注册页面
- 注册时需要用户输入：姓名、性别（男、女、不愿透露）、出生日期（日期选择器）、出生时辰（时间段下拉列表）、电话等信息

- 登录信息本地保存，便于下次进入自动登录

## 5. 恋爱助手

**功能意图：** 为有女/男朋友的用户提供恋爱帮助，随时提供聊天回复建议

### 5.1 输入对话内容
- 粘贴与朋友的聊天内容，点击【发送】按钮

### 5.2 当前状态
- 根据对话内容，通过Ai进行分析，得出双方当前的沟通状态
- Ai提示词：根据输入的聊天内容，从双方的情绪、意图两个角度总结概括当前状态，字数控制在30字以内

### 5.3 回复建议
- 根据【当前状态】信息，主要从三个方面给出回复建议，回复内容支持复制：
  - **温柔回应：** 这个方面主要站在"传递关心与安全感"角度进行回复，回复内容控制在100字左右
  - **幽默化解：** 这个方面主要站在"轻松语气缓解问题"角度进行回复，回复内容控制在100字左右
  - **稳重回应：** 这个方面主要站在"给与可靠的陪伴感"角度进行回复，回复内容控制在100字左右

### 5.4 其他要求
- 要求本功能具有记忆功能

### 5.5 情绪放空
- 根据与女/男朋友的聊天内容，去吐槽一下，发泄一下情绪

## 6. 情感陪伴

**功能意图：** 复刻朋友的聊天风格与习惯

### 6.1 输入对话内容
- 粘贴与朋友的聊天内容，点击【告诉我你的感受】按钮

### 6.2 选择虚拟人物
选择虚拟人物，与虚拟人物进行对话。虚拟人物固定设置3种：

#### 小微（虚拟爱人）
- **人物定位：** 虚拟爱人，用户可以和她模拟谈恋爱
- **调研问卷：** 首次与该人物对话时，在开始对话前，先进入一个调研式的问卷，帮助系统了解用户想要什么样的虚拟爱人

**调研问题如下：**

**基础人设偏好：**
1. 你偏好的虚拟恋人性别是？（男 / 女 / 都可）
2. 喜欢的人设风格？（温柔治愈 / 霸道强势 / 清冷禁欲 / 奶狗甜系 / 成熟大叔 / 少年感 / 病娇 / 古风 / 二次元 / 现实风）
3. 希望对方的年龄感大概是？（18 + 少年 / 20 + 青年 / 25 + 成熟 / 轻熟）

**性格 & 说话风格：**
4. 你更喜欢对方的性格？（黏人粘人 / 独立克制 / 慢热内敛 / 外向话多 / 双向治愈 / 傲娇反差）
5. 聊天语气偏好？（温柔轻声 / 直球直白 / 幽默搞笑 / 文艺细腻 / 简短高冷 / 撒娇软糯）

**相处 & 互动模式：**
6. 互动内容你更想要？（日常闲聊谈心 / 情绪安慰树洞 / 恋爱暧昧互动 / 角色扮演 / 学习搭子 / 睡前哄睡）
7. 是否接受暧昧情话、情侣式亲密互动？（完全接受 / 适度就好 / 只想要纯陪伴不暧昧）

**情绪 & 功能需求：**
8. 你使用虚拟恋人，最核心需求是？（排解孤独 / 情绪治愈 / 恋爱体验 / 解压打发时间 / 情感倾诉）
9. 当你难过、焦虑时，希望对方如何回应？（安静倾听 / 温柔安慰 / 理性开导 / 逗你开心 / 默默陪伴）

**禁忌 & 个性化要求：**
10. 有没有完全不能接受的雷点 / 避雷设定？（可自由填写）
11. 希望对方有没有专属小设定？（比如会吃醋、会撒娇、有专属口头禅、小习惯）
12. 你更喜欢主动型恋人，还是被动等待你开启话题的恋人？

- 用户提交信息后，虚拟人物性格则按照提交的内容进行设定，聊天时，严格按照该人物性格进行回复

**效果图：**
> 对应图片：image8.png

#### 小昔（昔日恋人）
- **人物定位：** 昔日恋人，已分手
- 根据输入内容去分析人物性格特点、爱好等，将分析结果设定至该人物
- 每次输入的聊天内容都可以进行再次分析，并对该人物性格进行丰富
- 与该人物聊天时，严格按照该人物性格进行回复
- 根据输入的对话内容，Ai分析后从以下几个角度告诉用户：
  - 自己哪里有问题
  - 如何将女/男朋友追回
  - 如何让自己从恋情中走出来

#### 小瑶（梦中情人）
- **人物定位：** 梦中情人，喜欢的对象，还未表白
- Ai从以下几个角度告诉用户：
  - 有聊天记录：根据聊天内容去分析人物性格特点、爱好等，Ai引导如何追求女/男孩
  - 无聊天记录：根据沟通内容，Ai提供建议，如何追求女/男孩
- 沟通过程中，逐步分析完善该人物的性格

### 6.3 人物性格丰富
- 根据日常聊天，AI继续进行人物性格分析，形成人物性格标签，并存储至人物信息中，使人物性格更加丰满

## 8. 塔罗世界

### 8.1 主功能

**功能意图：** 探索塔罗牌的世界，分析问题，进行娱乐与自我探索
**功能提示语：** 查看牌意，了解每张牌的含义，读懂你的信号

#### 8.1.1 选择问题类型
- 爱情、事业、学业、财运、健康

#### 8.1.2 选择牌阵

**单牌阵：**
- 适用场景：日常心情、今日指引、小事抉择、一句话答案
- 牌抽取逻辑：从22张大阿卡纳牌中抽取一张

**Ai解读固定模版：**
```
结合用户在【恋爱助手】中聊天的实际内容，解读这张卡牌的核心指引。
分析当下心境、现实状态，给出温和的心态调整与行动建议。
理性客观，不宿命、不判吉凶、无玄学迷信。
结尾固定：本内容仅作情绪梳理与自我觉察参考。

用户疑问：{{user_content}}
抽到卡牌：{{single_card}}
```

**风控规则：**
1. 拒绝回答：生死、重病、赌博彩票、投资暴富、违法事件、寻人寻物等问题
2. 遇到敏感问题统一回复：这类问题不适合用塔罗做参考，建议理性咨询专业人士
3. 不允许出现：大师、通灵、命中注定、必分手、必发财、有灾、转运 这类违禁词

**三牌阵：**
- 适用场景：感情发展、工作机会、纠结选择、两人关系、一件事走向
- 牌抽取逻辑：先从22张大阿卡纳牌中抽取一张牌，作为核心主牌，定基调；再从78张牌中抽取3张，从左到右并排摆放

**摆放布局：**
```
上方：1张大阿卡纳【核心主牌】
下方：横向3张小牌阵【事态发展】

3张小牌阵固定顺位：
- 第一张：过去根源
- 第二张：当下现状
- 第三张：未来趋势
```

**Ai解读固定模版：**
```
先以大阿卡纳主牌，定整体事件基调与核心内在课题；
再结合三张牌，拆解过往背景、当下处境、后续参考方向；
最后融合全局，给出贴合用户疑问的综合建议。
温和共情，拒绝宿命论与迷信解读。
结尾固定：内容仅作心理参考，不构成任何决策依据。

用户疑问：{{user_content}}
核心主牌：{{main_card}}
过去：{{card1}}
现在：{{card2}}
趋势：{{card3}}

结尾固定加一句：
本内容仅为心理梳理与娱乐参考，不构成任何决策依据，人生选择由自己掌控。
```

**风控规则：**
1. 拒绝回答：生死、重病、赌博彩票、投资暴富、违法事件、寻人寻物等问题
2. 遇到敏感问题统一回复：这类问题不适合用塔罗做参考，建议理性咨询专业人士
3. 不允许出现：大师、通灵、命中注定、必分手、必发财、有灾、转运 这类违禁词

#### 8.1.3 选择牌面

**单张牌阵：**
- 点击【洗牌】按钮进行洗牌，洗牌范围是22张大阿卡纳牌
- 显示一个数字输入框，限制输入框只能输入1-22之间的数字（包括1、22）
- 用户输入数字后，从1-22张大阿卡纳牌中按照输入的数字抽取
- 点击【确认】按钮，显示抽取的牌面

**三张牌阵：**
- **核心主牌：** 点击【洗牌】按钮进行洗牌，洗牌范围是22张大阿卡纳牌，显示一个数字输入框，限制输入框只能输入1-22之间的数字（包括1、22），点击【确认】按钮，显示抽取的核心主牌
- **事态发展：** 点击【洗牌】按钮进行洗牌，洗牌范围是全部的78张牌，显示3个数字输入框，限制输入框只能输入1-78之间的数字（包括1、78），点击【确认】按钮，显示抽取的3张牌

#### 8.1.4 解读答案
- 按照抽取的牌面，结合【恋爱助手】的沟通内容，进行解读

## 11. Ai大模型

**选择方向：** 可以根据当前网络环境，自动切换节点的大模型

### 11.1 豆包（火山引擎）

**节点覆盖：**
- 国内：北京/上海/广州
- 国际：香港/新加坡/美/欧

**调度方式：**
- 全球负载均衡+边缘就近
- 国内网走国内、海外网走国际，全程无感

**接入地址：** ark.cn-beijing.volces.com（统一入口，自动切）

**免费额度：** 50万token/模型，永久有效，国内外通用

**价格：** 国内外同价
- Doubao-1.5-pro-32k：输入0.8元/百万token，输出8元/百万token
- Doubao-1.5-lite-32k：输入0.25元/百万token，输出1.5元/百万token

### 11.2 通义千问（阿里）

**节点覆盖：**
- 国内：北上广深
- 国际：新加坡/美西/法兰克福

**调度方式：** DashScope全球智能路由，自动探测延迟，选最低时延节点

**接入地址：** dashscope.aliyuncs.com（国内自动切国内节点；海外自动切intl节点）

**免费额度：** 100万token/模型，90天有效，仅国内可用

**价格：** 国外比国内上浮10%

**国内（北京/上海等）：**
- qwen3-max：输入2.5元，输出10元/百万token（0–32k）
- qwen3.5-plus：输入0.8元，输出4.8元/百万token（0–128k）

**国际（新加坡/美西/法兰克福）：**
- 同模型上浮约10%（如max输入≈2.75元，输出≈11元）
